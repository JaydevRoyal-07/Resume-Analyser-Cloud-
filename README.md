# AI Resume Analyzer (Python stack)

This repo contains a Python-based resume analyzer:
- Backend: FastAPI
- ML Analyzer: Flask
- Frontend: Streamlit
- Database: MongoDB
- Orchestration: Docker Compose

Quick start:
1. Copy `.env.example` to `.env` and set `JWT_SECRET`.
2. Run: `docker compose up --build`
3. Open Streamlit UI at http://localhost:8501
